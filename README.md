# FISA-INFO-QCM
Streamlit pour s'entrainer sur des QCM de cours

Pour lancer en local le streamlit, il faut installer un environnement virtuel avec venv et installer streamlit pour executer l'app.

# 1. Crée le venv
python3 -m venv venv

# 2. Active-le
source venv/bin/activate

# 3. Installe streamlit
pip install streamlit

# 4. Lance l'app
streamlit run app.py
